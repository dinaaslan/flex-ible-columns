# flex-ible-columns
In this assignment I used different flex values to control how much space each column takes inside the row. 
First , I applied display: flex to .row which places all the child divs I have in a horizontal row. Then I added flex:1 to .col so each column gets equal share of space (each div becomes ¼). Then I added flex: 2 to .col-2 making it take twice as much space as .col (.col is ⅓ and .col-2 is ⅔) As for my last row, I added flex: 3; to .col-3 so it takes 3 parts of the space (.col-2 is ⅖ and .col-3 is ⅗)
